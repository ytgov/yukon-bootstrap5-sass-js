Modèle pour Bootstrap 5 | Système de conception pour Yukon.ca

Donnez à votre service l’apparence de Yukon.ca.

Commencez avec ce modèle pour inclure nos styles de base dans le code de votre application. Ajoutez ou supprimez des composants selon vos besoins.

Les styles CSS utilisés dans le système de conception, notamment les couleurs et les tailles, ne doivent pas être modifiés.

Consultez le Guide de prestation de services numériques y compris la section Modèles de conception en ligne, pour plus de détails sur l'utilisation de ces composants.

Cette version du système de conception Yukon.ca est basée sur Bootstrap 5. Consultez la documentation des exemples Sass et JS de Bootstrap 5 pour plus de détails.
